A este proyecto lo estoy haciendo para aprender a realizar apps web con Javascript siguiendo 
los pasos de este video: https://www.youtube.com/watch?v=-bI0diefasA.

Al desarrollo lo he llevado a cabo en Visual Studio Code y para ejecutar la app, debes tener 
Instalados: nodejs, npm y mongodb.

1 - Descargar o clonar los archivos.
2 - En una consola, ingresar a la carpeta donde se encuentran los archivos.
3 - Ejecutar el comando: "npm install".
4 - Iniciar mongodb (si lo haces por consola, debe ser en una nueva).
5 - Ejecutar el comando: "npm run dev".
6 - Por último, en un navegador web en: "http://localhost:3000/" podrás ver la app.

** Por ahora solo hay unas rutas de enlace con un texto simbólico: "/about", "/notes",
"/users/signin", "/users/signup".

En este commit estuve viendo como iniciar un proyecto desde 0 y a configurar la parte inicial 
del Backend, temas nuevos para mí.
